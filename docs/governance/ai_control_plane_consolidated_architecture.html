Hard invariants — không bao giờ bị vi phạm

🔒
core/policies.py không bao giờ bị thay bằng OSS PolicyEngine. Có thể tham khảo patterns từ ryanwi/agent-control-plane, nhưng class PolicyEngine và ConditionEvaluator luôn là code của bạn. Không import OSS policy runtimes.
🔒
core/models.py sở hữu toàn bộ data contracts. OSS types chỉ được dùng qua adapter wrappers. TelemetryEvent, PolicyDecision, AgentIdentity, McpError được định nghĩa đây và chỉ đây.
🔒
mcp/git_server.py là YOUR facade. cyanheads TypeScript fork là implementation bên dưới. Không có Git logic nào trong Python layer — chỉ policy dispatch và telemetry emission.
🔒
cli/ subcommands không bao giờ import OSS runtimes trực tiếp. cli/assign.py, cli/approve.py chỉ gọi core/ và api/. Không import FastMCP, CrewAI, hay OSS runtime nào khác.
🔒
apex/ sở hữu SAPAL loop. OSS anomaly tools (OTel IForest, Argos, Darts) được invoke từ apex/sense.py, apex/analyze.py, apex/predict.py — không bao giờ ngược lại.
🔒
api/ là cầu nối cross-language duy nhất. TypeScript PolicyClient gọi api/server.py HTTP endpoints. Không có channel khác (shared DB, message queue). Fail-closed: nếu api/server.py unreachable → TS server DENY tool call.
🔒
core/quota.py backend swappable, không hardcode. QuotaStore ABC cho in-memory (Milestone A), Redis (Milestone B), hoặc database — không đụng interface. Không import Redis ở module level.
🔒
config/ ở root = shipped defaults. Runtime config path resolve từ ACP_CONFIG_DIR env var. Loader không hardcode path. Tests dùng conftest.py fixture set ACP_CONFIG_DIR sang temp directory.
